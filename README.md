# Startup Lab

Startup Lab is my public product laboratory.

This is my digital playground where I build in public—every new idea gets its own live subdomain.

## What It Includes

- Personal landing page with founder-focused positioning
- Experiments/projects showcase
- Blog and blog detail pages
- About and contact pages
- Admin panel for blogs, projects, tags, profile, and messages
- Public terminal command palette

## Tech Stack

- Next.js
- TypeScript
- TypeORM
- MySQL
- Tailwind CSS
- Framer Motion

## Local Development

```bash
cd startup-lab
npm install
npm run dev
```

## Notes

- Main app code lives inside `startup-lab/`
- Messages submitted from the contact form are stored in the database and visible in the admin messages page
